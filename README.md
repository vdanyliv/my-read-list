## My read list

###### How to Install:
`npm install`

###### Mongo config
Please add this code into file ./server/config/mongo.js
```
module.exports = {
 'secret': 'your super secret key',
 'url': 'mongodb://your-mongo-url'
};
```
Please replace default value

###### How to Run:
`npm run dev` - for developing

`npm run prod` - for production
